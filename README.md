# Dart Tracker

A lightweight Flutter app for tracking personal dart training statistics.

## Features

### Score Tracking
- Track 180, 171 and 162 scores
- Track High Finishes
- Track Short Legs
- Interactive statistics chart
- Configurable baseline values
- Configurable colors
- Date range filters
- History
- Swipe to delete

### Finish Tracking
- Dedicated Finishes page
- Double and Triple finish tracking
- Finish distribution chart
- Date range filters
- Finish history
- Swipe to delete
- Bull disabled in Triple mode

### Backup
- Export and import all data
- Includes settings
- Includes score history
- Includes finish history

### UI
- Light & Dark Mode
- Customizable colors
